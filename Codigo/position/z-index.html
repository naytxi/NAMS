z-index

La propiedad z-index controla qué elementos se muestran encima o debajo de otros cuando se superponen. Funciona solo en elementos que tengan una propiedad position diferente de static.


Ejemplo:

contenedor div
position: relative

position: absolute;
top: 0rem;
left: 0rem;
z-index: 3;
